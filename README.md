### 📌 개념 정리  
✅ **차원(Dimension)**  
- 정성적 데이터(제품명, 날짜, 지리명 등)  
- 분석 기준이 되는 값  
- 불연속형 데이터로 측정값을 쪼개어 보는 하나의 관점  

✅ **측정값(Measure)**  
- 정량적 수치  
- 연속형 데이터로 집계가 되는 데이터
