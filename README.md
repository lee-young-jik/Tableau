# Tableau
태블로 신병훈련소 26기 
1일차
사용 데이터: 스타벅스 메뉴 데이터 & 매장 정보 데이터

    * 차원 : 정성적 데이터(제품명, 날짜, 지리명 등), 분석 기준이 되는 값, 불연속형 데이터로 측정값을 쪼개어 보는 하나의 관점
    * 측정값 : 정량적 수치, 연속형 데이터로 집계가 되는 데이터
